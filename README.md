Ferri Maintainer (v1.0)
🛠 What is it?

Ferri Maintainer is a lightweight utility designed for automated system maintenance. It performs a three-stage optimization:

    Security: Scans for active malware using Microsoft's MRT.

    Storage: Cleans temporary user and system directories to free up disk space.

    Integrity: Checks and repairs Windows system files via SFC and DISM modules.


📅 When should I use it?

It is recommended to run this utility once a month to keep the system healthy, though you can use it whenever you feel the system is underperforming.
